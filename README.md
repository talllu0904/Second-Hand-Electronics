# Second-Hand Electronics

A full-stack ecommerce platform for buying, selling, and verifying second-hand computer parts. This prototype includes product grading, verification workflows, part-compatibility checks, a build-saving tool, seller dashboards, and admin/technician management tools.

## Project Overview
This project provides a trusted marketplace for used computer components, with features like condition grading, testing/certification logs, return/refund modules, and compatibility checking.

## Features
- **User-Facing**: Account creation, search/filters, compatibility checker, PC build saving (CSV), cart, trade-in/returns.
- **Seller Features**: Revenue dashboard, inventory management, certification requests, subscriptions.
- **Admin/Technician**: Approve listings, manage disputes/returns, grading criteria, logs, analytics, account/item management.

## Architecture
- **Frontend**: HTML, CSS, JS (modular, themed).
- **Backend**: Node.js + Express (modular routes).
- **Database**: Mock in-memory (SQL schema for reference).
